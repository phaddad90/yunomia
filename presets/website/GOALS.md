## Standing Orders

- Every page must be tested at 375px, 768px, and 1440px widths
- No page ships without meta title, description, and OG tags
- Shared components go in a components/ directory, not duplicated
- Flag any third-party dependency that adds > 50KB to the bundle
- Content changes and code changes are separate tasks
